Detailed Description

Search schedules by name.

Request:

  • 'partial_name': A part of the name we are looking for.

Response: A list of {id,name} for doors that match the partial name. [ {id: $SCHEDULE_ID, name: $SCHEDULE_NAME}, {...} ]
