Detailed Description

Query the audit log.

Request:

  • enabled_types: An array of string representing the complete C++ type of the object. Possible types are:
    • Leosac::Audit::UserEvent
    • Leosac::Audit::WSAPICall
    • ... If enabled type is not present, returns all types.
  • p: Page number
  • ps: Page size

Response:

  • data: [JSON API data]
  • meta:
    • count: The number of entries that match the request.
    • totalPage: The number of page for to retrieve all items that match the request.
