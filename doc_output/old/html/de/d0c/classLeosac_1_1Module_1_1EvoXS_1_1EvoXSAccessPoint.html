Detailed Description

An AccessPoint that is backed by an EvoXS cylinder.

This object references a base_ap_ object, which is a Leosac::Auth::AccessPoint object. They share the same database identifier.

bool Leosac::Module::EvoXS::EvoXSAccessPoint::need_update_
private

Did we detect that we need a configuration update?

If this is false, it means we need no update UP TO the audit log pointed by checkpoint_. If checkpoint_ points to the last audit entry ever, then we are fine and need no update.
