13 Welcome to the main documentation page for the Rpleth module.
14 This page is about Rpleth from an end-user point of view. \n

16 
17 This module is for internal use at Islog, it's unlikely you will ever need it.
18 
19 Configuration Options {#mod_rpleth_user_config}
20 ===============================================
21 
22 Below are the configuration options available.
23 
24 Options | Description | Mandatory
25 -----------|--------------------------------------------------------|-----------
26 port | Which port should we bind to ? | YES
27 reader | Name of the wiegand reader the module handles | YES
28 stream_mode| If enabled, should forward to client all card read | NO (default to true)
29 
30 Example {#mod_rpleth_example}
31 -----------------------------
32 
33 This is a example of Rpleth possible configuration for Rpleth module into Leosac:
34 
35 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~.xml
36 <module>
37  <name>RPLETH</name>
38  <file>librpleth.so</file>
39  <level>100</level>
40  <module_config>
41  <port>4242</port>
42  <reader>MY_WIEGAND_1</reader>
43  <stream_mode>true</stream_mode>
44  </module_config>
45 </module>
46 ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
