Building Leosac

Describe how to build Leosac from source.

Target platform

This guide target a Debian Stretch system running on a x64 platform.

Dependencies

Leosac as a number of dependencies that needs to be installed:

apt-get install cmake build-essential git \
libssl-dev \
libcurl4-openssl-dev libtclap-dev libscrypt-dev \
python3 python3-pip

Database stuff

apt-get install libsqlite3-dev libpq-dev \
odb libodb-dev libodb-boost-dev libodb-sqlite-dev libodb-pgsql-dev

Boost libraries (todo)

apt-get install libboost1.62-all-dev

Build

Clone the repo: git clone https://github.com/leosac/leosac.git Now go to the leosac directory.

git submodule init && git submodule update
mkdir build;
cd build;
cmake ..
make -j10
