Enumeration Type Documentation

◆ APIStatusCode

enum Leosac::APIStatusCode
strong

Those are the Leosac API status code.

They currently apply only to the WebSocket API.

Note
This enum is provided by core because the audit system relies on it.
Enumerator
SUCCESS 

Indicate that the request's processing went well.

GENERAL_FAILURE 

A failure for an unknown reason.

PERMISSION_DENIED 

The websocket connection is not allowed to make the requested API call.

RATE_LIMITED 

The websocket connection is rate limited, and it already sent too many packets.

MALFORMED 

The source packet was malformed.

INVALID_CALL 

The API method (ie, message's type) does not exist.

TIMEOUT 

The request took too long to process.

This is mostly here as a placeholder, as this status_code will mostly be used internaly by the Javascript web app to signal a lack of response.

SESSION_ABORTED 

The session has been aborted.

This is likely due to the expiration of the token used to authenticate.

ENTITY_NOT_FOUND 

The requested entity cannot be found.

DATABASE_ERROR 

An internal database operation threw an exception.

UNKNOWN 

Unknown status.

Mostly useful as a default value somewhere.

MODEL_EXCEPTION 

Some internal API rules has failed.

This is likely to model validation.

INVALID_ARGUMENT 

One of the argument of the call had a invalid value / type.
