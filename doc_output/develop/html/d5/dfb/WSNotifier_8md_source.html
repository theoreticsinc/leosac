6 @brief A webservice client that will notify webservices of local events.

10 Introduction {#mod_ws-notifier_intro}
11 =======================================
12 
13 This modules listen to configurable authentication sources and forward the
14 authentication credentials to some configured web-services.
15 
16 Currently, only auth source event of type `Leosac::Auth::SourceType::SIMPLE_WIEGAND` are
17 handled, all the other are simply ignored.
18 
19 The card information is sent in an HTTP POST request.
20 
21 The POST field is `card_id` and represents the card id, in decimal.
22 
23 Configuration Options {#mod_ws-notifier_user_config}
24 ====================================================
25 
26 Options | Options | Options | Description | Mandatory
27 ---------------|----------|-----------------|----------------------------------------------------------------|-----------
28 want_ssl | | | Do we enable the SSL engine ? | NO (defaults to `true`)
29 sources | | | Multiples message source (wiegand reader) | NO
30 ---> | source | | Name of one reader to watch for event | YES
31 targets | | | Remote webservice to send information to | NO
32 ---> | target | | Describe a webservice server target. | NO
33 ---> | ---> | url | A complete URL that will be used as the request's destination. | YES
34 ---> | ---> | connect_timeout | The timeout (in milliseconds) for the connection phase. See `CURLOPT_CONNECTTIMEOUT_MS` for more information | NO
35 ---> | ---> | request_timeout | The timeout for the request, in milliseconds. See `CURLOPT_TIMEOUT_MS`. | NO
36 ---> | ----> | ca_file | Path to a PEM encoded CA file used to validate certificate. | NO
37 ---> | ---> | verify_host | If SSL is enabled, do we verify the host name in the SSL certificate ? | NO (defaults to `true`)
38 ---> | ---> | verify_peer | If SSL is enabled, do we verify the SSL certificate ? | NO (defaults to `true`)
39 
40 @note
41 The `connect_timeout` and `request_timeout` defaults to 7000 milliseconds.
42 
43 @note
44 `want_ssl` defaults to true, meaning you'll be able to contact webservice over HTTPS.
45 You can prevent the module from loading the SSL engine by setting this to `false`.
