10 Introduction {#mod_auth_db_intro}
11 ===================================
12 
13 The module Auth-DB is an **authentication backend** module.
14 It listens to events generated by **authentication source** module (like `Wiegand`). These events
15 contains credentials. This module verify the credential using Leosac's database, and either
16 grant or deny access.
17 Granting or denying access means sending a message to the application. This module doesn't open a door
18 or perform action on its own.
19 
20 @note Obviously this module requires that Leosac run with a database enabled.
