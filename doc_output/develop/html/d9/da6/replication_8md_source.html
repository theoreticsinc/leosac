10 Introduction {#mod_replication_intro}
11 =================================
12 
13 Leosac can run in a master/slave configuration. The master is not aware of his role, and
14 does not need to run any specific module.
15 
16 The slave unit needs the `Replication` module to automatically performs the
17 synchronization.
18 
19 @note Since synchronizing with a untrusted master in a huge security risk,
20 the slave needs the master's public key to make sure it talks to the right server.
21

23 Configuration Options {#mod_replication_user_config}
24 ====================================================
25 
26 Options | Description | Mandatory
27 -----------|--------------------------------------------------------|-----------
28 delay | Number of seconds between replication attempt. | NO, default to 120.
29 endpoint | Endpoint of the master server. | YES
30 pubkey | Public key of the master server | YES
